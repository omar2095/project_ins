Description

Professional Smart-Watch Health Monitoring System with real-time Bluetooth connectivity, GPS tracking, emergency alerts, and comprehensive health tracking. Built as a Progressive Web App (PWA) for seamless mobile and desktop experience.

Features

  • Real-time health monitoring (Heart Rate, SpO₂, Temperature)
  • Bluetooth Low Energy (BLE) connectivity
  • GPS tracking and location services
  • Emergency alert system with voice notifications
  • Offline mode with local data storage
  • Session recording and data export
  • Multiple theme options
  • Progressive Web App (PWA) support

Technologies

HTML5 CSS3 JavaScript (ES6+) Chart.js Leaflet.js Web Bluetooth API Firebase PWA
